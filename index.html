⚙️ ตั้งค่า Firebase ครั้งแรก

เพื่อให้ระบบ sync real-time ได้ ต้องใส่ค่า Firebase Config ของคุณใน code ก่อนนะครับ ทำตามขั้นตอนนี้:

1
ไปที่ console.firebase.google.com แล้วสร้าง Project ใหม่
2
ไปที่ Build → Realtime Database แล้วกด "Create database" เลือก Start in test mode
3
ไปที่ Project Settings (⚙️) → Your apps → Add app → Web แล้วคัดลอก firebaseConfig
4
เปิดไฟล์ HTML นี้ด้วย text editor แล้วแทนที่ค่า YOUR_... ในส่วน firebaseConfig ด้านบน
const firebaseConfig = {
  apiKey:            "AIzaSy...",
  authDomain:        "my-project.firebaseapp.com",
  databaseURL:       "https://my-project-default-rtdb.asia-southeast1...",
  projectId:         "my-project",
  storageBucket:     "my-project.appspot.com",
  messagingSenderId: "123456789",
  appId:             "1:123456789:web:abc123"
};
⏳ กำลังเชื่อมต่อ Firebase...
